üöÄ Quick Start Guide

Method 1: Recording Network Traffic

  1. Start Recording: Click the XHRScribe icon and press the Record button
  2. Interact with Website: Use the web application normally to trigger API calls
  3. Stop Recording: Click Stop when done
  4. Generate Tests: Go to Generate tab, configure options, and click "Generate Tests"

Method 2: Import Existing Data

  1. Import Files: Go to Import tab
  2. Choose Format: Drag & drop or select HAR, Postman, OpenAPI, or Insomnia files
  3. Preview & Import: Review the detected requests and click "Import Session"
  4. Generate Tests: Use the imported session in the Generate tab

AI Provider Setup

Configure your AI provider in the Settings (gear icon):

  • OpenAI: Requires API key from platform.openai.com - Supports gpt-4.1, gpt-4.1-mini, GPT-4-turbo

üéØ Best Practices

Recording Sessions

  • Plan Your Flow: Map out the user journey before recording
  • Clean Sessions: Start with a fresh browser session for cleaner data
  • Complete Workflows: Record entire user flows, not just individual requests
  • Name Sessions: Use descriptive names like "User Registration Flow" or "Order Checkout Process"

Import Best Practices

  • File Organization: Organize your HAR/collection files with descriptive names
  • Preview First: Always preview imported data before creating sessions
  • Batch Import: Import multiple related files for comprehensive test coverage
  • Metadata Preservation: Import preserves original metadata for better organization

Test Generation

  • Start Simple: Begin with basic tests, then add complexity
  • Review Generated Code: Always review and customize the generated tests
  • Use Custom Auth Guide: Provide detailed authentication instructions in settings
  • Endpoint Selection: Exclude non-critical endpoints to focus on important API calls

üîß Troubleshooting

Recording Issues

  • No Requests Captured: Check if the site uses supported request types (XHR/Fetch/GraphQL)
  • Missing Requests: Adjust filtering settings to include more request types
  • Permission Errors: Ensure debugger permissions are granted to the extension

Import Issues

  • File Format Errors: Ensure files are in valid JSON format
  • Large Files: Break down large collections into smaller chunks
  • Encoding Issues: Ensure files are UTF-8 encoded

Generation Issues

  • API Key Errors: Verify your AI provider API key is correct and has sufficient credits
  • Timeout Errors: Try using individual processing mode for complex sessions
  • Settings Recovery: Use the new reset options if experiencing persistent issues
